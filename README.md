# Football Tracker

A simple Next.js app for tracking football matches and stats.

## Requirements

- Node.js 18+
- yarn

## Setup

Install dependencies:

```bash
npm install
# or
yarn install
```

## Run the app

Start the development server:

```bash
npm run dev
# or
yarn dev
```

Open http://localhost:3000 in your browser.

## Build for production

```bash
npm run build
npm run start
```

## Lint

```bash
npm run lint
```
